# My Project — GM Hartshorne Fleet App

A Progressive Web App for **G. & M. Hartshorne Ltd** — driver management, timesheets, defect reports, PODs, and office dispatch.

## Quick Start

Open `index.html` in your browser, or serve with any static server:

```bash
npx -y serve .
```
